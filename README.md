# Nox - eCommerce Website

Nox is a fully responsive, beautifully designed eCommerce website built with HTML, CSS, and JavaScript. It features maximum compatibility across all mobile and desktop devices.

## Features

- **Fully Responsive Design**: Adapts seamlessly to any screen size.
- **Modern UI/UX**: Clean aesthetic with intuitive navigation.
- **Mobile-First Approach**: Optimized for mobile users.
- **Cross-Browser Compatibility**: Works consistently across major web browsers.

## Prerequisites

Before you begin, ensure you have met the following requirements:

* A modern web browser (Chrome, Firefox, Safari, Edge, etc.)
* [Git](https://git-scm.com/downloads "Download Git") (Optional, for cloning the repository)

## Getting Started

To get a local copy up and running, follow these simple steps:

### Installation

1. Clone the repo:
   ```bash
   git clone https://github.com/Nox-25/nox-ecom.git
   ```

2. Navigate into the project directory:
   ```bash
   cd nox-ecom
   ```

3. Open `index.html` in your preferred web browser to view the site.
